# Project 1: Weather Storyteller

## Task 1: Choose a Free Weather API and Get Your API Key

For this project, we'll use **OpenWeatherMap**, one of the most popular and beginner-friendly free weather APIs.

### Steps to Get Your API Key:

1. Go to [OpenWeatherMap](https://openweathermap.org/api).
2. Click **Sign Up** and create a free account.
3. Once logged in, navigate to the **API Keys** section in your profile.
4. Copy your default API key (or generate a new one if needed).
